# Contribution

## Main Contribution

- 뉴스 기반 텍스트 파생변수 생성
- 키워드 사전 기반 변수 생성
- LDA 토픽 기반 변수 생성
- FinBERT 임베딩 기반 변수 생성
- 텍스트 변수 포함 전후 성능 비교
- 텍스트 변수가 최종 F2-score를 개선하지 못해 제외된 과정 정리
- 타깃 정의, 변수 설계, 불균형 처리, 모델 평가에 대한 팀 논의 참여

## What This Means

텍스트 데이터가 직관적으로는 디페깅 위험을 설명할 수 있을 것이라 예상했지만, 실험 결과 최종 모델의 F2-score 개선으로 이어지지 않았습니다. 이 결과를 숨기지 않고, 최종 모델에서 텍스트 변수를 제외한 판단 과정을 정리했습니다.
